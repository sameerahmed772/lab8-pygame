Project Snapshot

This project builds a real-time moving-square simulation with size-based speed limits, random acceleration, predator-prey style fleeing, wall bounce, and click-to-score interactions. The game loop in main.py follows a clean event/update/render cycle and uses typed constants for readability and tuning.

Runtime Behavior

Each frame: input events are consumed, square physics is updated unless paused, then scene and overlays are rendered. Difficulty modifies acceleration and effective speed cap.

Core controls: SPACE add square, P pause, 1/2/3 difficulty, left click remove square and add score.
